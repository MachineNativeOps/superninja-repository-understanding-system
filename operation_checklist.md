# 操作檢查清單系統

## 操作前必須執行的檢查

### 1. 上下文驗證
- [ ] 確認當前工作目錄
- [ ] 確認目標檔案/目錄的存在性
- [ ] 確認檔案的當前狀態
- [ ] 確認檔案的用途和重要性

### 2. 影響評估
- [ ] 識別此操作會影響的其他檔案
- [ ] 評估操作風險等級
- [ ] 確認是否有備份或版本控制
- [ ] 確認是否需要通知相關方

### 3. 知識檢查
- [ ] 我是否理解此檔案的完整用途？
- [ ] 我是否知道此檔案的依賴關係？
- [ ] 我是否確認此操作符合整體架構？
- [ ] 我是否有足夠的資訊進行此操作？

### 4. 驗證步驟
- [ ] 在測試環境中驗證操作
- [ ] 檢查操作結果的輸出
- [ ] 驗證沒有意外副作用
- [ ] 確認系統狀態正常

## 禁止操作的情況
- ❌ 不清楚檔案用途時
- ❌ 不確定影響範圍時
- ❌ 沒有備份時
- ❌ 在生產環境未測試時

## 緊急停止機制
當發現以下情況時立即停止：
- 操作結果不符合預期
- 發現未預期的錯誤訊息
- 系統行為異常
- 發現對檔案用途的誤解
